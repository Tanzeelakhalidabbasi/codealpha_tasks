# CodeAlpha — UI/UX Design Internship Tasks

This repository contains my completed tasks for the **UI/UX Design Internship** at CodeAlpha.

**Intern:** Tanzeela Khalid
**Domain:** UI/UX Design
**Project:** Social Connect — a minimal social media app

---

## About the Project

Social Connect is a minimal social media application designed with a focus on
clean visuals, ease of use, and a calm, clutter-free experience. The design was
built in Figma and follows core UI/UX principles: clear visual hierarchy,
consistent navigation, accessible colour contrast, and one primary action per
screen.

---

## Tasks Completed

### Task 1 — Wireframing & Low-Fidelity Design
Low-fidelity wireframes for five core screens of the Social Connect app:
Login, Home Feed, User Profile, Post Creation, and Settings. Built as simple
grey-box layouts in Figma to establish structure before visual design.

### Task 2 — High-Fidelity UI Design
High-fidelity versions of the screens with a cohesive colour palette,
typography, icons, and components. Includes a complete Style Guide page
documenting colours, type scale, spacing, and component states.

### Task 3 — UX Case Study
A case study analysing the Zomato food delivery app — its user journey,
strengths, and weaknesses — with three proposed UX improvements.
See: `Zomato_UX_Case_Study.pdf`

### Task 4 — Mini Project: Prototype Design
A clickable, connected prototype of the Social Connect app built in Figma,
with all screens linked to demonstrate the full user flow. A written
explanation of the design process is included.
See: `Prototype_Design_Explanation.pdf`
